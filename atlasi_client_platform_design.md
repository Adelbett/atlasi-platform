خلفية `#050505` مع **جسيمات ذهبية** طائرة ببطء (canvas animation) + نسيج حبوب طفيف. في المنتصف، صورة مظلة كبيرة تلمع ببطء مع **توهج ذهبي** يدور حولها.

### التأثيرات المميزة في Hero

**1. صورة المظلة — 3D Parallax Float:**
```jsx
// الصورة تتبع حركة الماوس بخفة
const handleMouseMove = (e) => {
  const x = (e.clientX / window.innerWidth - 0.5) * 15;
  const y = (e.clientY / window.innerHeight - 0.5) * 10;
  imageRef.style.transform = `
    perspective(1000px) 
    rotateY(${x}deg) 
    rotateX(${-y}deg)
    translateZ(20px)
  `;
};
```

**2. جسيمات الخلفية — Golden Dust:**
```
- 60 جسيم صغير (2-4px) ذهبية/بيج
- تطفو لأعلى ببطء مع تلاشٍ تدريجي
- تُولد بشكل عشوائي كل 200ms
- Canvas API — خفيف على الأداء
```

**3. الرقم العداد — Count Up:**
```
عند الوصول للشاشة: الأرقام تعدّ من صفر
  247+ مشروع منجز
  5    سنوات ضمان
  24   ساعة خدمة
```

**4. نص Hero — Staggered Reveal:**
```
الحروف تظهر واحدة واحدة بتأخير 30ms بين كل حرف
مع انزلاق من الأسفل + تلاشٍ — Cairo 900 Bold
```
